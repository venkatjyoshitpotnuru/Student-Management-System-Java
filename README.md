# Student Management System - Java

A menu-driven Student Management System built using Java, OOP concepts, ArrayList, Linear Search, Sorting, Git, and GitHub.

## Features

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Sort Students by Marks
- Sort Students by Name

## Technologies Used

- Java
- OOP
- ArrayList
- Linear Search
- Sorting
- Git
- GitHub
