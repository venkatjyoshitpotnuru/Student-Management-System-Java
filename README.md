# Student-Management-System-Java
A menu-driven Student Management System built using Java, OOP concepts, ArrayList, Linear Search, and Git version control.
A console-based Student Management System developed using Java.

Features:
- Add Student
- View Students
- Search Student by ID
- Menu Driven Interface
- ArrayList Data Storage
- Linear Search Implementation

Concepts Used:
- Object-Oriented Programming (OOP)
- Classes and Objects
- Constructors
- Encapsulation
- ArrayList
- Linear Search
- Loops and Conditional Statements
- Git & GitHub

Technologies:
- Java
- VS Code
- Git
- GitHub
